# react-memo

## 설명

이 프로젝트는 React를 이용한 메모 앱입니다.

&nbsp;

## 목표

- 모달을 통해 메모를 추가하거나 수정할 수 있습니다.
- 메모 목록에서는 원하는 메모를 삭제할 수 있습니다.
